# Glitch Market

Glitch Market es un e-commerce desarrollado con React como proyecto final de Coderhouse.

## Demo online

[Ver proyecto en Vercel](https://glitch-market-proyectofinal-fernandez-alfredo-4aflnagyw.vercel.app)

La aplicación permite:

- Ver un catálogo de videojuegos
- Filtrar productos por plataforma
- Ver el detalle de cada juego
- Agregar productos al carrito
- Simular una compra mediante checkout
- Guardar productos y órdenes en Firebase

## Tecnologías utilizadas

- React
- React Router DOM
- Firebase / Firestore
- Vite
- CSS

## Funcionalidades principales

- Listado dinámico de productos desde Firestore
- Filtrado por categorías
- Vista de detalle de producto
- Carrito de compras con Context API
- Checkout con generación de orden
- Deploy online con Vercel
